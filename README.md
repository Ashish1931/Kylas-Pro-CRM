# Kylas Pro Site Setup

This is a premium, high-converting landing page built and integrated with Kylas CRM.

## Installation Steps
1. **Visit the URL**: Go to `http://localhost/kylas-pro-site` in your browser.
2. **Finish WP Install**: Enter your site title, username, and password.
3. **Log in to Dashboard**.
4. **Activate Theme**: Go to **Appearance > Themes** and activate **"Kylas Premium Theme"**.
5. **Activate Plugins**:
   - Go to **Plugins > Installed Plugins**.
   - Activate **Contact Form 7**.
   - Activate **WP Mail SMTP** (and configure your SMTP account).
   - Activate **Kylas CRM Integration** (Our custom plugin).
6. **Create Your Form**:
   - Go to **Contact > Contact Forms**.
   - Create a form with names like `first-name`, `last-name`, `email`, `phone`, `city`, `company`.
7. **Map Fields**:
   - Go to **Kylas CRM > Field Mapping**.
   - Select your form and map the fields to Kylas fields.

## Key Features
- **Sticky Header**: Includes a "Contact Us" CTA button that scrolls to the form.
- **Glassmorphism Design**: Modern, premium look with dark mode aesthetics.
- **Advanced Lead Tracking**: Captured data is saved locally in JSON format and synced to Kylas CRM.
- **Automated Notifications**: Sends HTML emails to both Admin and Lead upon success.
